# AAA
A future project buddy (Portfoilio)

Technology Stack
เทคโนโลยีที่ใช้
- AI / Machine Learning: ใช้โมเดลในการจำแนกรูปแบบพระแท้–ปลอม
- OpenCV: ใช้สำหรับประมวลผลภาพและตรวจหาตำหนิในแต่ละส่วนของพระเครื่อง
- Frontend/UI Design: ออกแบบด้วยแนวทางสมัยใหม่ (Modern Interface) เน้นการใช้งานง่าย
- Result Analytics: แสดงผลการวิเคราะห์เป็นเปอร์เซ็นต์ความถูกต้อง (Accuracy Score) และคะแนนรวม (Predictive Score)

Key Features
จุดเด่นของโปรเจ็ค
- วิเคราะห์พระเครื่องอัตโนมัติด้วย AI
- แสดงจุดตำหนิของพระแบบ Realtime
- ประเมินผลคะแนนความแท้และความสมบูรณ์
- ออกรายงานรับรอง (Certificate) อัตโนมัติ
- ใช้งานง่ายผ่านระบบ Login/Register
